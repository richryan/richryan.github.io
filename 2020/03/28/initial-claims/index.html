Initial Claims

March 28, 2020

Here are initial-claims data, which were covered in a recent NY Times article. I try and replicate a few of the figures detailed in the article.

Data on initial claims are available at FRED by browsing data by Category > Population, Employment, & Labor Markets > Weekly Initial Claims and selecting Initial Claims.

Here is the R code for retrieving and plotting these data. Data on initial claims from March are highlighted in orange.

# Load packages
library(tidyverse)
library(tidyquant)
library(ggplot2)
library(ggthemes)

my_blue <- "purple1"
my_orange <- "orangered1"

# Convert claims to millions
dat <- claims %>% 
  mutate(price = price / 1e+06)

dat_early <- dat %>% 
  filter(
    date >= ymd("2004-01-01") &
    date <  ymd("2020-03-01"))

dat_late <- dat %>% 
  filter(date >= ymd("2020-03-01"))

last_date <- max(dat_late$date)
last_lvl <- dat_late$price[[which(dat_late$date == max(dat_late$date))]]

my_breaks <- seq(0,7)
my_labels <- paste(my_breaks)
my_labels[[length(my_labels)]] <- "7 million"

plt <- ggplot() +
  geom_col(mapping =  aes(x = date, y = price),
           data = dat_early,
           color = my_blue, position = "identity") +
  # Highlight March 2020
  geom_col(data = dat_late,
             mapping = aes(x = date, y = price),
             color = my_orange) +
  xlab("") + ggtitle("Initial jobless claims") +
  # Label y-axis for wsj theme
  scale_y_continuous(breaks = my_breaks,
                     labels = my_labels, 
                     limits = c(0,7.5)) +
  # Label last observation, set off 1 year
  annotate("text", 
           x = last_date - 360,
           y = last_lvl, 
           label = paste0(last_lvl, "\n", 
                         month(last_date, label = TRUE), ", ", day(last_date)),
           color = my_orange,
           hjust = 0.85) +
  # Use sans serif font instead of mono, use gray color
  theme_wsj(base_family = "sans", 
            color = "gray", 
            title_family = "sans")
# Display the plot
(plt)

FRED also provides recession dates. These dates can be used to add recession shading to the developing figure.

The goal is highlighting recessions. From the NBER’s website:

Contractions (recessions) start at the peak of a business cycle and end at the trough.

Note: The FRED series USRECM indicates periods “from the peak through the trough”; whereas, USREC indicates periods “from the period following the peak through the trough.” (Both series are monthly.)

recess <- tq_get("USRECM", get = "economic.data", from = "1800-01-01")

To use geom_rect(), the beginning and end of recessions are needed.

# Create groups based recessions
recess_dat <- recess %>% 
  arrange(date) %>% 
  mutate(same = 1 - (price == lag(price))) %>% 
  # Remove first row, an NA, for cumulative sum
  filter(date > min(recess$date)) %>% 
  mutate(era = cumsum(same)) %>% 
  # Filter only recessions
  filter(price == 1)

Now the recessions are grouped according to era. The following code groups the data by recessions and keeps the first and last element of the group.

recess_dat <- recess_dat %>% 
  group_by(era) %>% 
  # Unncessary, but to be sure...
  arrange(date) %>% 
  filter(row_number() == 1 | row_number() == n())

# Now reshape the data wide.
# Each row will contain the start and end dates of a recession.
recess_dat <- recess_dat %>% 
  mutate(junk = row_number()) %>% 
  mutate(begin_end = case_when(
    junk == 1 ~ "begin",
    junk == 2 ~ "end"
  ))

recess_wide <- recess_dat %>%
  ungroup() %>% 
  select(symbol, price, date, era, begin_end) %>% 
  pivot_wider(names_from = begin_end, values_from = date)

Adding recession shading is straightforward:

plt2 <- plt + 
  geom_rect(mapping = aes(xmin = begin, xmax = end,
                          ymin = 0, ymax = 7),
            data = filter(recess_wide,
                          begin >= min(dat_early$date)), 
            fill = "blue",
            alpha = 0.2)
(plt2)
