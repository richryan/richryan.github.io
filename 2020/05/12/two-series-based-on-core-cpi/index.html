Two Series Based on Core CPI

May 12, 2020

Here are two series based on the Consumer Price Index for All Urban Consumers: All Items Less Food and Energy in US City Average.

From FRED:

The “Consumer Price Index for All Urban Consumers: All Items Less Food & Energy” is an aggregate of prices paid by urban consumers for a typical basket of goods, excluding food and energy. This measurement, known as “Core CPI,” is widely used by economists because food and energy have very volatile prices.

Tidying data

To calculate some of the statistics reported in the Consumer Price Index Summary, FRED publishes formulas used to transform data series.

# Load packages
library(tidyverse)
library(tidyquant)
library(ggplot2)
library(ggthemes)

# Chart parameters
mygray <- "gray66"

# Data for recession shading
recess <- tq_get("USRECM", get = "economic.data", from = "1800-01-01")
tail(recess)

## # A tibble: 6 x 3
##   symbol date       price
##   <chr>  <date>     <int>
## 1 USRECM 2019-11-01     0
## 2 USRECM 2019-12-01     0
## 3 USRECM 2020-01-01     0
## 4 USRECM 2020-02-01     0
## 5 USRECM 2020-03-01     0
## 6 USRECM 2020-04-01     0

recess_dat <- recess %>% 
  arrange(date) %>% 
  mutate(same = 1 - (price == lag(price))) %>% 
  # Remove first row, an NA, for cumulative sum
  filter(date > min(recess$date)) %>% 
  mutate(era = cumsum(same)) %>% 
  # Filter only recessions
  filter(price == 1)

recess_dat <- recess_dat %>% 
  group_by(era) %>% 
  # Unncessary, but to be sure...
  arrange(date) %>% 
  filter(row_number() == 1 | row_number() == n())

# Now reshape the data wide.
# Each row will contain the start and end dates of a recession.
recess_dat <- recess_dat %>% 
  mutate(junk = row_number()) %>% 
  mutate(begin_end = case_when(
    junk == 1 ~ "begin",
    junk == 2 ~ "end"
  ))

recess_wide <- recess_dat %>%
  ungroup() %>% 
  select(symbol, price, date, era, begin_end) %>% 
  pivot_wider(names_from = begin_end, values_from = date)

# Consumer Price Index for All Urban Consumers: All Items Less Food and Energy in U.S. City Average
core <- tq_get("CPILFESL", get = "economic.data", from = "1800-01-01") 
core <- core %>% 
  mutate(change = 100*(price/lag(price) - 1),
         change12 = 100*(price/lag(price, n = 12) - 1))
# Drop NAs
core <- core %>% 
  drop_na(change)

# These numbers should agree with the published BLS summary
tail(select(core, date, change, change12))

## # A tibble: 6 x 3
##   date       change change12
##   <date>      <dbl>    <dbl>
## 1 2019-11-01  0.194     2.32
## 2 2019-12-01  0.125     2.25
## 3 2020-01-01  0.242     2.27
## 4 2020-02-01  0.223     2.37
## 5 2020-03-01 -0.103     2.10
## 6 2020-04-01 -0.448     1.44

According to the release: “The index for all items less food and energy fell 0.4 percent in April, the largest monthly decline in the history of the series, which dates to 1957,” which agrees with the above number listed last in the change column. And: “The index for all items less food and energy increased 1.4 percent over the last 12 months, its smallest increase since April 2011,” which also agrees with the above number listed last in the change12 column.

Percent change from one year ago

Here is the chart for percent change from one year ago:1

ggplot(data = core) +
  # Add recession shading
  geom_rect(data = filter(recess_wide,
                          begin >= min(core$date)), 
            mapping = aes(xmin = begin, xmax = end,
                          ymin = -Inf, ymax = Inf),
            fill = mygray, alpha = 0.7) +
  geom_line(mapping = aes(x = date, y = change12), 
            size = 0.8, color = "black") +
  ggtitle("12-month percent change in core CPI") +
  # Make sure start at 0%
  expand_limits(y = 0) +
  # Use sans serif font instead of mono, use gray color
  theme_wsj(base_family = "sans",
            color = "gray",
            title_family = "sans",
            base_size = 10)

One-month percent change

And here is the chart for one-month percent change, April 2019–April 2020:

core1yr <- core %>% 
  filter(date >= ymd("2019-04-01")) %>% 
  mutate(mnth = month(date, label = TRUE, abbr = TRUE),
         yr = round(100*(year(date)/100 - round(year(date)/100, 0))),
         my_x = factor(paste0(mnth, " '", yr))) %>% 
  mutate(my_x = fct_reorder(my_x, date))

ggplot(data = core1yr) +
  geom_col(mapping = aes(x = my_x, y = change)) +
  theme_wsj(color = "gray", 
            base_family = "sans", 
            title_family = "sans",
            base_size = 8) +
  ggtitle("One-month percent change in core CPI")

  1. Note: It is probably more appropriate to use the not-seasonally-adjusted series to compute the 12-month percent change. ^
