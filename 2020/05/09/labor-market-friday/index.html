Labor Market Friday

May 9, 2020

Here are data from the latest Employment Situation. The charts are based on this NY Times article, which begins “The American economy plunged deeper into crisis last month, losing 20.5 million jobs as the unemployment rate jumped to 14.7 percent, the worst devastation since the Great Depression.”

The R code shared below replicates some of the charts included in the article. The charts are listed in the table of contents.

It should be noted that the BLS published a statement on how the Coronavirus impacted the data from the April 2020 Establishment and Household Surveys. While the collection rate for the Establishment Survey was “essentially unchanged” from collection rates prior to the pandemic (partly due to a longer collection period), the Household Survey response rate “at about 70 percent, was about 13 percentage points lower than in months prior to the pandemic.”

As David Blanchflower put it for The New York Review of Books: “The data are hard to interpret because any idea of ‘normal’ has gone out of the window.” Blanchflower points out that the data from the Establishment Survey does not include the self-employed.

Monthly change in jobs

The calculation of the monthly change in jobs uses data from the Establishment Survey. From the BLS: “In the establishment survey, workers who are paid by their employer for all or any part of the pay period including the 12th of the month are counted as employed, even if they were not actually at their jobs. Workers who are temporarily or permanently absent from their jobs and are not being paid are not counted as employed, even if they are continuing to receive benefits. The length of the reference period does vary across the respondents in the establishment survey; one-third of businesses have a weekly pay period, slightly over 40 percent a bi-weekly, about 20 percent semi-monthly, and a small amount monthly.”

# Load packages
library(tidyverse)
library(tidyquant)
library(ggplot2)
library(ggthemes)

# Chart parameters
mygray <- "gray66"

# Data for recession shading
recess <- tq_get("USRECM", get = "economic.data", from = "1800-01-01")
tail(recess)

## # A tibble: 6 x 3
##   symbol date       price
##   <chr>  <date>     <int>
## 1 USRECM 2019-11-01     0
## 2 USRECM 2019-12-01     0
## 3 USRECM 2020-01-01     0
## 4 USRECM 2020-02-01     0
## 5 USRECM 2020-03-01     0
## 6 USRECM 2020-04-01     0

recess_dat <- recess %>% 
  arrange(date) %>% 
  mutate(same = 1 - (price == lag(price))) %>% 
  # Remove first row, an NA, for cumulative sum
  filter(date > min(recess$date)) %>% 
  mutate(era = cumsum(same)) %>% 
  # Filter only recessions
  filter(price == 1)

recess_dat <- recess_dat %>% 
  group_by(era) %>% 
  # Unncessary, but to be sure...
  arrange(date) %>% 
  filter(row_number() == 1 | row_number() == n())

# Now reshape the data wide.
# Each row will contain the start and end dates of a recession.
recess_dat <- recess_dat %>% 
  mutate(junk = row_number()) %>% 
  mutate(begin_end = case_when(
    junk == 1 ~ "begin",
    junk == 2 ~ "end"
  ))

recess_wide <- recess_dat %>%
  ungroup() %>% 
  select(symbol, price, date, era, begin_end) %>% 
  pivot_wider(names_from = begin_end, values_from = date)

# All Employees, Total Nonfarm
payems <- tq_get("PAYEMS", get = "economic.data", from = "1800-01-01") 
payems <- payems %>% 
  # Convert to millions
  mutate(price = price / 1000,
         change_jobs = price - lag(price),
         mycat = date == max(date)) %>% 
  filter(!is.na(change_jobs))

payems <- payems %>% 
  left_join(recess, by = "date") %>% 
  mutate(neg = change_jobs < 0)
  
last_date <- max(payems$date)
last_lvl <- payems$change_jobs[[which(payems$date == last_date)]]

segment_height <- 1.5

# Set up labels for y axis
my_breaks <- c(seq(1, -2), seq(-5, -20, by= -5))
my_labels <- c("+1 million", 
               paste(c(seq(0, -2), seq(-5, -20, by= -5))))

# Plot
ggplot(data = payems) +
  geom_col(mapping = aes(x = date, y = change_jobs,
                         color = neg, fill = neg)) +
  scale_y_continuous(breaks = my_breaks,
                     labels = my_labels) +
  # Color the bars
  scale_fill_manual(values = c(mygray, "red")) +
  scale_color_manual(values = c(mygray, "red") ) + 
  # Label last observation 
  annotate("text", x = last_date - 12*100, y = 3.0, 
           label = paste(last_lvl, "million"), 
           color = "red") +
  annotate("segment", x = last_date, xend = last_date,
           y = 0, yend = 2.5,
           linetype = "dashed",
           color = "black") +
  # Turn off legend
  guides(color = FALSE, fill = FALSE) +
  ggtitle("Monthly change in jobs") +
  # theme_pander() 
  theme_wsj(base_family = "sans",
            color = "gray",
            title_family = "sans",
            base_size = 8) +
  # Put grid lines on top
  theme(panel.background = element_rect(fill = NA),
        panel.ontop = TRUE)

Unemployment rate

Data used to calculate the unemployment rate come from the Current Population Survey or Household Survey. The Household Survey classifies people as employed, unemployed, or not in the labor force “based on their answers to a series of questions about their activities during the survey reference week (April 12th through April 18th).”

The unemployment rate or number of unemployed as a percentage of the labor force is depicted below.

ur <- tq_get("UNRATE", get = "economic.data", from = "1800-01-01")
ur <- ur %>% 
  mutate(series = "Unemployment rate")

# Last observation
last_date <- max(ur$date)
last_lvl <- ur$price[[which(ur$date == max(ur$date))]]

# Scale
my_breaks <- seq(2, 14, by = 2)
my_labels <- paste(seq(2, 12, by = 2))
my_labels <- c(my_labels, "14%")

ggplot(data = ur) +
  # Add recession shading
  geom_rect(data = filter(recess_wide,
                          begin >= min(ur$date)), 
            mapping = aes(xmin = begin, xmax = end,
                          ymin = -Inf, ymax = Inf),
            fill = mygray, alpha = 0.7) +
  geom_line(mapping = aes(x = date, y = price), 
            size = 0.8, color = "black") + 
  # Highlight last observation
  geom_point(data = filter(ur, date == last_date), 
             mapping = aes(x = date, y = price),
             size = 3) +
  xlab("") + ylab("Percent") + ggtitle("Unemployment rate") +
  # Label y-axis for wsj theme
  scale_y_continuous(breaks = my_breaks,
                     labels = my_labels) +
  # Make sure start at 0%
  expand_limits(y = 0) +
  # Label last observation
  annotate("text", x = last_date, y = last_lvl + 1, 
           label = paste0(last_lvl, "%"),
           color = "black") +
  xlab("") + ylab("Percent") + ggtitle("Unemployment rate") +
  # Use sans serif font instead of mono, use gray color
  theme_wsj(base_family = "sans",
            color = "gray",
            title_family = "sans",
            base_size = 8)

Share of the population that is employed

The BLS’s Employment Situation Summary reports: “The employment–population ratio, at 51.3 percent, dropped by 8.7 percentage points over the month. This is the lowest rate and largest over-the-month decline in the history of the series (seasonally adjusted data are available back to January 1948).”

Here is the employment–population series:

Share of unemployed on temporary layoff—some hope

There is some hope—with the caveat that the BLS classified unemployed respondents who couldn’t answer when they expected to be recalled to work as unemployed on temporary layoff.

The unemployed includes “all persons who had no employment during the reference week, were available for work, except for temporary illness, and had made specific efforts to find employment some time during the 4 week-period ending with the reference week. Persons who were waiting to be recalled to a job from which they had been laid off need not have been looking for work to be classified as unemployed.”

For April 2020, the reference week was from April 12 through April 18th. “Workers who indicate they were not working during the entire survey reference week and expect to be recalled to their jobs should be classified as unemployed on temporary layoff.”

Temporary layoff is one of the categories used by the BLS to classify the unemployed. The BLS uses the following categories:

  • Job losers and persons who completed temporary jobs
    • On temporary layoff
    • Not on temporary layoff
      • Permanent job losers
      • Persons who completed temporary jobs
  • Job leavers
  • Reentrants
  • New entrants

The chart in this section depicts the share of unemployed on temporary layoff. To be classified as unemployed on temporary layoff, a person has either been given a date to return to work by their employer or expects to be recalled to their job within 6 months. From the BLS: “If, because of the coronavirus, a person was uncertain of when they would be able to return to work and thus was unsure of how to answer the question, the interviewer was instructed to enter a response of ‘yes,’ rather than ‘don’t know.’ This would allow the individual to be included among the unemployed on temporary layoff.” In other words, this was “an attempt to classify people who were effectively laid off due to pandemic-related closures among the unemployed on temporary layoff.”

Some unemployed people on temporary layoff may also have received pay from their employer: “Under the guidance provided to the household survey interviewers, workers who indicate that they were not working during the entire reference week due to efforts to contain the spread of the coronavirus should be classified as unemployed on temporary layoff, whether or not they are paid for the time they were off work.”

With this caveat, from the NY Times article: “It remains possible that the recovery, too, will be swift, and that as the pandemic retreats, businesses that were fundamentally healthy before the virus will reopen, rehire and return more or less to normal. The one bright spot in Friday’s report was that nearly 80 percent of the unemployed said they had been temporarily laid off and expected to return to their jobs in the coming months.”

loss_series <- c("LNS13023653", "UNEMPLOY")
                  
loss <- tq_get(loss_series, get = "economic.data", from = "1900-01-01")

# Rename for ease
loss <- loss %>%
  mutate(series_name = case_when(
    symbol == "UNEMPLOY" ~ "unemp",
    symbol == "LNS13023653" ~ "layoff")) %>% 
  select(-symbol)

# Reshape wide
loss <- loss %>% 
  pivot_wider(names_from = series_name,
              values_from = price)

# Create share of unemployed on temporary layoff
loss <- loss %>% 
  mutate(share = 100*layoff / unemp) %>% 
  drop_na(share)

# Last observation
last_date <- max(loss$date)
last_lvl <- loss$share[[which(loss$date == last_date)]]
last_lvl <- round(last_lvl, 1)

# Scale
my_breaks <- seq(10, 80, by = 10)

                          begin >= min(loss$date)),
