Many models in R

June 15, 2019

Introduction

This is an example of using R to run many regressions and collect a single regression coefficient from each model. The code uses R’s list-columns feature.

Generate inflation data as percent change from one year ago:

$$ \text{percent change from year ago} = \left( \frac{ x_t }{ x_{t-12} } - 1 \right) \times 100. $$

dat <- dat %>% 
  group_by(symbol) %>% 
  mutate(infl = (price/lag(price, n = 12) - 1)*100) %>% 
  drop_na()

# Rename series
dat <- dat %>% 
  mutate(series_name = case_when(symbol == "CPILFESL" ~ "Less food and energy",
                                 symbol == "CPIAUCSL" ~ "All items"))

Plot the data:

ggplot(data = dat) +

  labs(x = "", y = "Percent", color = "") + 
  ggtitle("Inflation") +
  theme_bw()

Regressions

The goal now is seeing the relationship between the unemployment rate and inflation. Specifically, we’re interested in the regressions $$ \text{inflation}_t = \alpha + \beta \; \text{unemployment rate}_t + \varepsilon_t $$ for each of the inflation series. We’re interested in reporting the $ \beta $s.

To do so, we need to get the data in this form:

dat <- dat %>%

Now we’re interested in a function that takes one of these data frames as inputs.

my_reg <- function(df) {
  fit <- lm(infl ~ ur, data = df)
  # Return the unemployment-rate coefficient
  return(fit$coefficients[["ur"]])
}

And we want to use R’s map function to return the coefficients:

dat <- dat %>% 
  mutate(coefficient = map(data, my_reg)) %>% 
  unnest(coefficient)
dat
