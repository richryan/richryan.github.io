$ \LaTeX{} $

Including inline $ \LaTeX{} $ math is done by wrapping expressions in a single $. Multiline expressions are wrapped in $$. For example,

$$ \begin{equation} \label{eq:some-eq} \mathbf{y} = \mathbf{X} \beta + \varepsilon, \end{equation} $$

which can be referred to as $ \eqref{eq:some-eq} $. Automatic equation numbering is accomplished by including the HTML code

<script type="text/x-mathjax-config">
MathJax.Hub.Config({
  TeX: { equationNumbers: { autoNumber: "AMS" } }
});
</script>

in a post. See http://docs.mathjax.org/en/latest/tex.html#automatic-equation-numbering.

Table of contents

A table of contents can be added to R Mardown posts by adding the following options to YAML:

output: 
  blogdown::html_page:
    toc: true

and using {{% toc %}} anywhere in the document.
